# Advanced-SQL-Data-Analysis

**🌟 Highlights**

This project demonstrates a comprehensive understanding and application of key SQL concepts and advanced functions through a real-world data analysis scenario.

*   🔍 **Exploratory Data Analysis (EDA)** using various queries.
*   📊 **Complex Querying** with joins and subqueries.
*   📈 **Data Aggregation** using `GROUP BY`, `HAVING`, and aggregate functions like `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`.
*   🔢 **Window Functions** (`ROW_NUMBER()`, `RANK()`, `LEAD()`, `LAG()`, `AVG()` over partition) for sophisticated analytical tasks.
*   ✅ **Conditional Logic** implemented with `CASE` statements for data categorization and transformation.
*   🛠️ **Data Integrity Checks** to ensure data quality and reliability.

## ℹ️ Overview

This repository contains SQL scripts used to analyze a dataset. The primary goal was to extract meaningful insights, generate reports, and showcase proficiency in advanced SQL techniques.

The project addresses questions like: 
"What are the top 5 performing products by sales region?" 
"How does customer acquisition change over time?"
"Which departments have an average salary above a certain threshold?"
The included scripts provide clear, commented examples of how these questions are answered using the specified SQL functions.  

The repository is organized into several `.sql` files, each focusing on specific concepts. Key scripts include:

*   `04_measures_exploration.sql`: Contains examples of `SELECT`, `WHERE`, `JOIN`, and basic filtering.
*   `05_magnitude_analysis.sql`: Demonstrates `GROUP BY`, `HAVING`, `COUNT`, `SUM`, etc.
*   `10_data_segmentation.sql`: Provides examples of conditional logic for data categorization.
*   `13_report_products.sql`: Combines all techniques to generate final reports and insights.
