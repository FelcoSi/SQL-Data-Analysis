# SQL-Data-Analysis

Transforming transactional data into performance insights using complex joins, CTEs, and window functions. The project computed change-over-time trends, cumulative metrics, and part-to-whole proportional analysis to evaluate growth patterns and operational efficiency.

A comprehensive collection of SQL scripts for data exploration, analytics, and reporting. Covers various analyses such as database exploration, measures and metrics, time-based trends, cumulative analytics, segmentation, and more.
